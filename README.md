# Telegram Test Bot

The bot is intended to be used for:

- testing your server environment;
- checking compatibility of libraries and runtime;
- verifying Telegram Bot API connectivity;
- testing general bot functionality.

## Usage

The project can be freely used, modified, and adapted for:

- development testing;
- learning and experimentation;
- infrastructure and deployment checks.
